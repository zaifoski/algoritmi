g++ -o correttore -O2 -Wall correttore.cpp

Ad esempio, se avete eseguito il vostro codice utilizzando i dati del caso 5, eseguite il seguente comando:
./correttore input/input5.txt output/output5.txt output_vostra_soluzione.txt
